注意：

以下是 PHP 中接口的一些注意事项：

  1. 接口中定义的所有方法都必须是公共方法（public），不能定义为私有方法（private）或受保护方法（protected）。
  2. 接口不能包含属性，只能包含方法的声明。
  3. 实现接口的类必须实现接口中所有的方法，否则会报错。
  4. 一个类可以同时实现多个接口，多个接口之间使用逗号（,）分隔。
  5. 接口也可以继承其他接口，通过使用 extends 关键字实现，可以继承多个接口。
  6. 如果一个类同时实现了多个接口，且这些接口中有同名的方法，那么这个类只需要实现一次该方法即可。
  7. 接口可以被类或抽象类实现，实现接口的类必须实现接口中所有的方法，而抽象类可以只实现部分方法，剩下的方法留给其子类去实现。
  8. 接口中的方法默认都是抽象的，不需要使用 abstract 关键字。

这段代码意思是将 $this->address 对象进行克隆，并将克隆出的对象赋值给 $this->address 变量。由于 $this->address 是一个对象，使用普通的赋值操作 $this->address = $this->address 只是让 $this->address 变量指向同一个对象，而不是创建一个新的对象。因此，为了确保 $this->address 指向的是一个全新的对象，需要使用 clone 关键字进行克隆。

如果我们把__clone()方法设置为private，那么就会禁止克隆操作。

十四、常用的魔术方法

总结

在 PHP 中，魔术方法是一种特殊的方法，以双下划线开头和结尾，用于在类中执行特定的操作或行为。以下是一些常用的魔术方法及其说明：

  1. __construct()：构造方法，在创建对象时自动调用，通常用于初始化对象的属性和方法。
  2. __destruct()：析构方法，在对象被销毁时自动调用，通常用于清理对象的资源和内存。
  3. __get()：获取不存在或无权访问的属性时自动调用，可以在该方法中进行自定义的处理和返回值。
  4. __set()：设置不存在或无权访问的属性时自动调用，可以在该方法中进行自定义的处理和赋值操作。
  5. __call()：调用不存在或无权访问的方法时自动调用，可以在该方法中进行自定义的处理和返回值。
  6. __callStatic():调用一个不存在的静态方法时自动调用。
  7. __toString()：将对象转换为字符串时自动调用，可以在该方法中返回自定义的字符串表示形式。
  8. __isset()：判断不存在或无权访问的属性是否已设置时自动调用，可以在该方法中进行自定义的判断和返回布尔值。
  9. __unset()：取消设置不存在或无权访问的属性时自动调用，可以在该方法中进行自定义的处理和清空操作。
  10. __clone()：对象克隆时自动调用，可以在该方法中进行自定义的处理和克隆操作。
  11. __invoke()：将对象作为函数调用时自动调用，可以在该方法中进行自定义的处理和返回值。
  12. __autoload()：当使用尚未被定义的类时自动调用，可以在该方法中根据类名进行类文件的自动加载和引入。
  13. __sleep():使用serialize序列化的时候自动调用。
  14. __wakeup():使用unserialize反序列化时自动调用。
  15. __set_state():使用var_export()函数时自动调用，接受一个数组参数。
  16. __debugInfo():使用var_dump()打印对象信息时自动调用。

这些魔术方法可以在 PHP 类中使用，使程序更加灵活和强大。

十七、$_SESSION和$_COOKIE

$_COOKIE和$_SESSION都是PHP中的超全局变量，用于在Web应用程序中存储和访问cookie和session数据。它们的使用方法如下：

$_COOKIE:

  1. 设置cookie：使用setcookie()函数，语法如下：

    setcookie(name, value, expire, path, domain, secure, httponly);

    name: cookie的名称。 value: cookie的值。 expire: cookie的过期时间，以Unix时间戳表示，默认为0，表示在浏览器关闭时过期。 path: 可访问该cookie的路径，默认为’/‘，表示整个网站都可以访问。 domain: 可访问该cookie的域名，默认为空字符串，表示只有当前域名可以访问。 secure: 是否只能通过HTTPS协议来传输cookie，默认为false。 httponly: 是否只能通过HTTP协议来访问cookie，默认为false。

    例如：setcookie(‘username’, ‘John Doe’, time()+3600, ‘/‘);

  2. 获取cookie：使用$_COOKIE超全局变量来获取cookie的值，例如：

    $username = $_COOKIE[‘username’];

  3. 删除cookie：设置cookie的过期时间为当前时间之前的时间即可删除cookie，例如：

    setcookie(‘username’, ‘’, time()-3600);

$_SESSION:

  1. 启动session：在使用$_SESSION之前，需要先启动session，可以通过session_start()函数来实现，例如：

    session_start();

  2. 设置session：可以通过给$_SESSION数组赋值来设置session的值，例如：

    $_SESSION[‘username’] = ‘John Doe’;

  3. 获取session：使用$_SESSION超全局变量来获取session的值，例如：

    $username = $_SESSION[‘username’];

  4. 删除session：使用unset()函数来删除session，例如：

    unset($_SESSION[‘username’]);

  5. 销毁session：可以通过session_destroy()函数来销毁session，例如：

    session_destroy();

文件判断

在PHP中，可以使用文件系统函数来获取文件的大小、是否可执行、是否可读、是否可写、文件创建时间、上次修改时间等信息。以下是常用的函数：

filesize()：获取文件大小，返回文件大小的字节数。

is_executable()：判断文件是否可执行，返回布尔值。

is_readable()：判断文件是否可读，返回布尔值。

is_writable()：判断文件是否可写，返回布尔值。

filectime()：获取文件创建时间，返回创建时间的时间戳。

filemtime()：获取文件上次修改时间，返回修改时间的时间戳。
