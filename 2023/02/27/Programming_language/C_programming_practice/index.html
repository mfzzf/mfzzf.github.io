P1005 辗转相除法求最大公约数

辗转相除法（又称欧几里得算法）是求两个正整数的最大公约数的一种算法。该算法基于以下定理：对于任意的两个正整数a和b（a>b），它们的最大公约数等于b和a%b的最大公约数。具体的实现过程如下：

  1. 如果b为0，则a就是最大公约数，递归结束；
  2. 否则，继续递归计算gcd(b, a % b)，并返回结果。

例如，假设要求24和60的最大公约数，按照辗转相除法，有：

  • gcd(24, 60) = gcd(60, 24) = gcd(24, 12) = gcd(12, 0) = 12。

因此，24和60的最大公约数为12。

辗转相除法的时间复杂度为O(log(max(a,b)))，其中max(a,b)表示a和b的最大值。这个时间复杂度相对较小，因此辗转相除法是求最大公约数的一种高效的算法。

  1. 如果n小于等于1，则直接返回false；
  2. 对于i从2开始，循环到sqrt(n)，判断n是否能被i整除；
  3. 如果能被整除，则说明n不是质数，返回false；
  4. 如果循环结束后仍然没有找到能整除n的数，则说明n是质数，返回true。

冒泡排序

冒泡排序是一种简单的排序算法，它的基本思想是通过不断比较相邻的两个元素，将较大（或较小）的元素交换到数组的一端，从而达到排序的目的。

具体来说，冒泡排序的过程如下：

  1. 比较相邻的两个元素，如果第一个元素比第二个元素大（或者小，具体实现可以根据需要而定），则交换这两个元素的位置。
  2. 对数组中的所有元素都执行上述操作，这样一次冒泡就完成了。
  3. 重复执行第一步和第二步，直到所有元素都排好序为止。

可以看出，冒泡排序的核心就是交换相邻的元素。每一轮冒泡都会将数组中最大（或最小）的元素交换到数组的一端，这样在下一轮冒泡中就可以不再考虑已经排好序的元素，因此每轮冒泡的范围都会缩小一些。

P1035 鸡兔同笼问题

这是一个经典的数学问题，可以通过解方程组来解决。假设笼子中鸡的个数为a，兔子的个数为b，则可以得到以下方程组：

a + b = m （1） # 头的个数为m

2a + 4b = n （2） # 腿的个数为n

将（1）式乘以2，然后用（2）式减去（1）式的两倍，得到：

2b = n - 2m

则有：

b = (n - 2m) / 2

将b代入（1）式，得到：

a = m - b

然后就可以求出鸡和兔子的个数了。

第n项在第几层呢？

只要找到一个最小的k，使得S≥n，那么n便在第k层。

第n项就是第k条斜线上倒数第S-n+1个数（最后一个元素是倒数第1个元素，而不是倒数第0个元素）。

而k的奇偶决定着第k条斜线上数的顺序：若k是奇数，第k条斜线上倒数第i个元素是i/(k+1-i)；若k是偶数，第k条斜线上倒数第i个元素是(k+1-i)/i。
