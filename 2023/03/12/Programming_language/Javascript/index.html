Javascript基础

简介

JavaScript是一种脚本语言(用户驱动型)，用于在Web浏览器中实现动态交互和网页效果。它是一种解释型语言，可以直接嵌入HTML代码中，并由浏览器执行。JavaScript支持面向对象、函数式和基于原型的编程范式，并提供了丰富的内置API和外部库，使其成为最流行的客户端脚本语言之一。除了浏览器外，JavaScript也可以在服务器端通过Node.js运行。

对象继承

对象继承是指在 JavaScript 中，一个对象（称为子对象）可以通过继承另一个对象（称为父对象）的属性和方法来扩展自己的功能。在 JavaScript 中，对象继承可以通过原型链实现。

每个 JavaScript 对象都有一个内部属性 [[Prototype]]，它指向该对象的原型。原型可以理解为是一个模板或蓝图，包含了一组共享的属性和方法。当我们访问一个对象的属性或方法时，如果该对象本身没有定义这个属性或方法，则会沿着原型链向上查找，直到找到该属性或方法为止。

闭包

闭包是指在 JavaScript 中，一个函数能够访问并操作定义在其外部作用域中的变量的能力。简单来说，闭包就是一个函数和它的相关变量组成的一个包裹体。

URI

URI 是用于表示资源标识符的字符串，包括 URL 和 URN。在 JavaScript 中，可以使用 encodeURI()、encodeURIComponent()、decodeURI() 和 decodeURIComponent() 四个函数来对 URI 进行编码和解码。

  • encodeURI()：对整个 URI 进行编码，但不会对本身属于 URI 的特殊字符进行编码，例如冒号、正斜线、问号和井号等。
  • encodeURIComponent()：与 encodeURI() 类似，但会对一些更多的字符进行编码，包括 URI 中的分隔符和保留字符等。
  • decodeURI() 和 decodeURIComponent()：分别对 encodeURI() 和 encodeURIComponent() 编码的 URI 进行解码，将编码后的字符串转换回原始字符串。

open函数

在 JavaScript 中，window.open() 函数可以打开一个新的浏览器窗口或标签页，并加载指定 URL 的页面。

window.open() 函数接受三个参数：

  1. URL：要加载的页面的 URL。
  2. target：指定新页面的打开方式，可以是以下四种取值之一：
    • _self：在当前窗口中加载页面（默认值）。
    • _blank：在新标签页中加载页面。

History和Location对象

history对象是浏览器窗口的历史记录，可以通过它来访问和操纵用户在当前会话期间访问过的网址。例如，使用history.back()方法可以返回到前一个页面，而使用history.forward()方法可以前进到下一个页面。

location对象表示当前加载页面的URL，并且可以用于获取或设置该URL的各个部分，例如路径名、主机名、端口号等。它还提供了一些方法和属性，如location.href可以用于获取或设置当前页面的URL，而location.reload()可以重新加载当前页面。

window.history对象用于访问和操纵浏览器窗口的历史记录。它提供了以下方法：

  • history.back(): 导航回到上一页。
  • history.forward(): 导航到下一页。
  • history.go(n): 导航到相对于当前页面的第n个页面，n可以是正数或负数。
  • history.pushState(stateObj, title, url): 向历史记录中添加一个新条目，并且改变当前URL，但不会导致页面重载。
  • history.replaceState(stateObj, title, url): 替换当前历史记录中的条目，而不会创建一个新的条目。

除了这些方法，history对象还有一些属性，例如length表示历史记录中的条目数量，state返回当前激活历史记录条目的状态对象等。通过使用window.history对象，开发者可以控制浏览器的历史记录，实现自定义的前进后退行为，以及在不刷新页面的情况下改变URL等操作。

window.location对象表示当前加载页面的URL，并且可以用于获取或设置该URL的各个部分，例如路径名、主机名、端口号等。它提供了以下属性和方法：

  • location.href: 返回或设置当前页面的完整URL。
  • location.protocol: 返回或设置当前页面的协议（http://或https://）。
  • location.host: 返回或设置当前页面的主机名和端口号。
  • location.hostname: 返回或设置当前页面的主机名。
  • location.port: 返回或设置当前页面的端口号。
  • location.pathname: 返回或设置当前页面的路径名。
  • location.search: 返回或设置当前页面的查询字符串部分。
  • location.hash: 返回或设置当前页面的哈希部分。
  • location.reload(): 重新加载当前页面。

通过使用window.location对象，开发者可以对当前页面的URL进行操作，例如重定向到其他页面、修改查询字符串参数等等。注意，使用location.href或location.replace()属性或方法时将会导致页面的刷新或跳转。
